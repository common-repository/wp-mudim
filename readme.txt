== Description ==
-EN:

A simple Vietnamese keyboard for typing Vietnamese languague on Wordpress. After actived it, users don't need any external Vietnamese keyboard program to type Vietnamese on WordPress.

This plugin is used a javavscript called "mudim" (http://code.google.com/p/mudim/) - a free solution for Vietnamese keyboard on website
-VI:

Đây là một plugin nhỏ tích hợp bộ gõ Mudim vào WordPress của bạn mà không cần phải chỉnh sửa template bằng tay. Bộ gõ có tác dụng trên cả Front-End và Back-End.

== Installation ==

1. Upload plugin-name to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done ! Enjoy it :-)
